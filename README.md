# Automotive Manufacturing Production Analysis (SQL)

## Project Overview

This project analyzes automotive manufacturing production data using SQL to identify inefficiencies and quality issues.

The analysis focuses on:

- Production trends
- Rejection rates
- Plant performance
- Product model performance
- Quality inspection outcomes

## Tools Used

SQL

## Database Structure

The project includes four relational tables:

1. production_data
2. plant_details
3. product_details
4. quality_inspection

## SQL Concepts Used

- Aggregations (SUM, AVG)
- JOINs
- Window Functions
- Subqueries
- CASE statements
- GROUP BY and HAVING

## Key Insights

- Identified plants with highest production output
- Detected models with higher defect rates
- Ranked plant performance using window functions
- Analyzed daily production trends
